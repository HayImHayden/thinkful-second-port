Hello. I'm Hyunjin a Web Developer!


Hello there. I’m Hyunjin. I was a student who was studying for a pre-nursing program in college, but I’m currently studying software engineering at Thinkful. Specifically, I’m working with HTML, CSS, and JavaScript for now.
